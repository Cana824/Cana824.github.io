Write a small story about HTML and CSS and their relationship. You could personify elements or speak from your own personal experience. The story should capture insight about the nature of HTML and CSS.

HTML is a builder - building up structures like walls, doors, frames and roofs. HTML lays the foundation of the working element before CSS - a designer can beautify everything. CSS changes color, make interactive elements which completes the build. CSS cannot exist without HTML constructing the base of which CSS can act upon- therefore a collaborative relationship.

Any moments of surprise or joy?

While learning CSS, I found a really useful shortcut using selectors like #album > img and body > *. Instead of styling each element one-by-one, I could write a single line of code to apply styles to multiple elements at once.

For example:
#album > img means “select all elements that are direct children of the element with id=album.” It ignores any images that are nested deeper inside other tags. I used this to CSS code all images within my id=album for the photos I added onto my blog.
body > * means “select all elements that are direct children of the .” The * is the universal selector—it represents any element, no matter the tag (like div, h1, section, etc.). I wanted to remove the weird spacing on the edges of the site so I made the body margin = 0px. However, this also caused all the words to start from the edge. To resolve this, I added padding using the above body > *. Thus, instead of writing code individually for each body element - I used the body > * coding to refer to all children within the body element.

If HTML and CSS were animals, which animals do you think they would be? Why?

If HTML and CSS were animals I would say they are Bowerbirds. CSS would be the male bowerbirds-they build nests that are pretty. These are decorated with colourful things like berries and flowers to attract mates.It’s all about appearance and style - just like CSS focuses on making web content visually appealing. HTML, on the other hand, would be the female bowerbird. She builds the actual functional nest- a soft, saucer-shaped structure high above the ground where she lays eggs. This is similar to how HTML creates the structure and content of a webpage. Both male and female bowerbirds are involved in nest-building, just in different, complementary ways-just like HTML and CSS work together to build websites. One creates the structure, and the other adds beauty and visual design.
